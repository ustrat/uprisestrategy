Uprise Strategy — Architecting Intelligence
A modern, multi‑page React + TypeScript website for Camir Consulting, showcasing services in BI optimization, AI/LLM advisory, Private AI for SMBs, and the AI Data Center Locator.

🚀 Overview
This project is a fully responsive, theme‑enabled website built with:

- React 18
- TypeScript
- Vite
- React Router
- Custom Dark/Light Theme System
- CSS‑based design system

It includes all core pages:

- Home  
- Services  
- About  
- Insights  
- Contact  

The site is designed to reflect your consultancy’s identity: technical authority, clarity, and modern intelligence.

🛠️ Getting Started

1. Install dependencies
`
npm install
`

2. Start the development server
`
npm run dev
`

3. Build for production
`
npm run build
`

4. Preview the production build
`
npm run preview
`

---

🎨 Theme System (Dark/Light Mode)

The site includes a full theme engine with:

- System preference detection  
- LocalStorage persistence  
- Smooth transitions  
- A toggle button in the navbar  

Themes are controlled via:

`
:root[data-theme="dark"]
:root[data-theme="light"]
`

You can customize colors in styles.css.

---

🌐 Routing

The site uses React Router v6 for client‑side navigation:

- / → Home  
- /services → Services  
- /about → About  
- /insights → Insights  
- /contact → Contact  

---

🧩 Customization

You can easily extend:

- Add new pages under src/pages/
- Add new components under src/components/
- Modify global styles in styles.css
- Update theme logic in ThemeContext.tsx

---

📦 Deployment

You can deploy the built site to:

- Netlify  
- Vercel  
- GitHub Pages  
- Azure Static Web Apps  
- Any static hosting provider  

Just upload the dist/ folder generated by:

`
npm run build
`
